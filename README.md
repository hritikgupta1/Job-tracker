# Job Tracker (PHP + MySQL)

A modern web application to track job applications efficiently.  
Built with PHP, MySQL, Bootstrap, and AJAX for a smooth user experience.

## ✨ Features

- 📊 Dashboard with job statistics
- ➕ Add, ✏️ Edit, ❌ Delete job entries
- 🔍 Filter jobs by:
  - Status (Applied, Interview, Offer, Rejected)
  - Company
  - Role
- 📝 Notes system with popup modal
- ⚡ AJAX-based job loading (no page reload)
- 📱 Fully responsive (Bootstrap 5)
- 🔐 Session-based authentication (Login/Register)

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, Bootstrap 5, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Other:** AJAX (Fetch API)
